# Technopoly

The main website for Technopoly event.

## How to install

1. Clone this repository:

```bash
git clone https://github.com/muohannedxd/technopoly.git
```

2. Go into project

```bash
cd technopoly
```

3. Install dependencies

```bash
npm install
# or
yarn install
```

4. Run project

```bash
npm run dev
or
yarn dev
```
